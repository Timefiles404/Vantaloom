# 本地素材库

这个目录用于暂存 Vantaloom UI 库后续可能用到的第三方素材、registry、许可证和来源记录。

## 目录

```text
assets/
  icons/
    nucleo-glass/          Nucleo Glass Icons 的公开 MIT 包快照
  motion/
    reactbits/             React Bits registry 快照
  ui-references/
    uiverse-ugly-lion-23/  Uiverse 毛玻璃卡片来源记录
  ux/
    shadcn/                shadcn/ui 候选组件 registry 快照
```

## 规则

- 每个来源必须保留许可证或来源说明
- 未确认授权的素材只记录链接和说明，不直接复制源码或二进制资产
- 商业授权素材需要补齐许可证文件后再进入可发布资产集
- 大体积预览视频、gif 和站点资源默认不进仓库
